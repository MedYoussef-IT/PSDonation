PSDonation is an Android application created to raise awareness and provide practical ways to support Palestinians.

The app includes historical information about Palestine, an interactive real map, and a curated list of verified donation links (free and paid). It combines education with action in a simple and informative interface.

## Features
- History of Palestine
- Interactive real map
- Verified donation links
- Awareness-focused content
- Simple and informative UI

## Technologies Used
- Java
- Android SDK

## Project Status
Completed

## Purpose
This project was built to combine mobile development skills with social and humanitarian awareness.
